# CLI Reference

ProteinMCP provides two command-line tools for managing MCP servers and workflow skills.

| Command | Purpose |
|---------|---------|
| [`pmcp`]({{ site.baseurl }}/cli/pmcp) | MCP server management — install, register, status, create |
| [`pskill`]({{ site.baseurl }}/cli/pskill) | Workflow skill management — install, uninstall, list |
