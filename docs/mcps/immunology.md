# Immunology MCPs

Predict MHC binding for peptide-based immunology applications.

| MCP | Runtime | Description |
|-----|---------|-------------|
| [NetMHCpan MCP](https://github.com/Biomolecular-Design-Nexus/netmhcpan_mcp) | python | NetMHCpan-4.2 for MHC Class I binding prediction |
| [NetMHCIIpan MCP](https://github.com/Biomolecular-Design-Nexus/netmhc2pan_mcp) | python | NetMHCIIpan-4.3 for MHC Class II binding prediction |

## Install

```bash
pmcp install netmhcpan_mcp
pmcp install netmhc2pan_mcp
```

{: .note }
NetMHCpan and NetMHCIIpan are free for academic/non-commercial use. The MCP servers download binaries from the official CBS source during installation.
